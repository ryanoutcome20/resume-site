pGui💻

pGui is a lightweight, modular GUI framework for Garry's Mod, written entirely in Lua. It provides customizable UI elements to integrate into your menu designs.

Features include:

  • Integration - Designed to seamlessly integrate with existing projects. Built with minimal setup and maintenance in mind.
  • Modular Architecture - Built to easily integrate new UI elements. Already existing elements can be easily detoured.
  • Portable - Zero dependencies besides the default built-in VGUI framework.
  • Retained Mode UI - Constantly stored, built for simplicity and memory efficiency.

Atlas🌐

Atlas is a networking library built for Source Engine games. It provides a stable framework in which reliable networking can be accomplished with more than the typical 64KB data transmission.

It provides several useful features including:

  • Modular Architecture - Drag-and-drop simplicity for easy integration.
  • Dynamic Hooking System - Allows multiple functions to be hooked to the same networking string for greater flexibility.
  • Lightweight - The entire library is made to be a low-drag, compression-heavy system with low networking overhead.
  • Integration - Designed to work seamlessly with existing projects without networking collisions, unlike other libraries.

Coffee☕

Coffee is an open-source trainer software for Garry's Mod designed to run modularly with various open-source C++-based modules. The entire framework was built mainly for anti-cheat developers who needed a reliable and stable test framework in which they could improve their detections.

Unique Features:

  • Modular Architecture - Built to run on any system architecture and any third-party module.
  • Secure Environment - Utilizes the setfenv feature inside of Lua to generate a secure execution environment.
  • Plugin System - Built with a plugin system that allows developers to add dynamically into the secure environment of the software.
  • Lightweight - The entire trainer is made to be a lightweight system with a custom hooking system, custom GUI system, and much more.
  • Customizable & Unique Visuals - Dozens of features never before seen in private or public trainer software. Including weather, dynamic stencil-rendered glow, fog modulation, sky modulation, and more.
  • Hitbox Manager - Dynamic hitbox manager, which dynamically parses hitbox sets and hitbox groups to select an ideal hitbox target when aiming.

Cryptify🔒

A simple yet elegant file encryption and decryption tool built with the C# .NET Framework and WinForms. It allows users to securely encrypt and decrypt files using different encryption algorithms, including AES, DES, and TripleDES.

Features include:

  • Selectable & Customizable Algroithm - Easily switch between AES, DES, or TripleDES to secure your files.
  • Simple UI - An easy to navigate and utilize interface.
  • Customizable Algroithms - Easily swap out encryption algorithms or extend the application with new methods.
  • Easily Built - For developers, the encryption system comes with an easy to use batch file to build it automatically.
  • Modular - For developers, the encryption system is modular and easy to integrate new algorithms into.

Zephyr🔥

An open-source game modification tool (trainer) for older versions of the game Counter-Strike: Global Offensive. Older versions were chosen to prevent users from utilizing the trainer against other real players. Made specifically to research injected GUI systems and VMT hooking.

Features include:

  • Dynamic VMT Hooking System - Easily detour virtual method table entries with your own function pointers.
  • UI System - Hijacked surface rendering system that is built into the Source Engine to render a dynamic GUI.
  • Rebuilt Animations - Rebuilt enemy animations to prevent interpolation under lag, which drastically improves accuracy.
  • Rebuilt WindowProc - Hooked WindowProc to enable dynamic input from binds or other input listeners.
  • Various Exploits - Various exploits including hitbox manipulation, lag manipulation, disable respawn positions, deathmatch godmode, etc.

Benchmark Toolkit⚖️

An open-source utility and development library for benchmarking Lua functions and code under specific conditions. Can produce statistics for both memory and timing.

Features include:

  • Dynamic Memory & Timing Statistics - Provides dynamic memory and timing statistics with easy-to-understand output.
  • Customizable Configuration Options - Easily modify configuration settings on the fly using console variables.
  • Easily Utilized Functions & Commands - Includes multiple API functions and an easy-to-use console command interface.
  • Clientside & Serverside - Available in both clientside and serverside modes.
  • Direct & Addon - Available in both direct execution and addon modes. See the addon branch for more information.
